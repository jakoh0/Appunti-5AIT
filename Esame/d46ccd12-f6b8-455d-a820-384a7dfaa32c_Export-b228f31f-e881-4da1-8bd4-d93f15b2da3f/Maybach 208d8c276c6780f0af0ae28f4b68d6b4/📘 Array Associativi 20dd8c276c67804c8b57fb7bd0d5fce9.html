📘 Array Associativi

📘 Array Associativi in PHP (tipo dizionario)

In PHP, esistono gli array associativi, che sono simili ai dizionari in altri linguaggi come Python.

🔹 Cosa sono?

Un array associativo è una struttura che collega una chiave a un valore.

📌 Invece di usare solo numeri (come gli array normali), puoi usare nomi o etichette come chiavi.

🧪 Esempio Base

$persona = [
    "nome" => "Mario",
    "eta" => 30
];

echo $persona["nome"]; // Stampa: Mario
echo $persona["eta"];  // Stampa: 30

In questo esempio:

  • "nome" è una chiave, e "Mario" è il suo valore.
  • "eta" è una chiave, e 30 è il suo valore.

🔧 Aggiungere o Modificare Valori

$persona["cognome"] = "Rossi"; // Aggiunge una nuova coppia
$persona["eta"] = 31;          // Modifica la chiave "eta"

🔁 Stampare tutto con foreach

foreach ($persona as $chiave => $valore) {
    echo "$chiave: $valore\\n";
}

🔸 Output:

nome: Mario
eta: 31
cognome: Rossi

🧠 Riepilogo

  • Un array associativo è come un dizionario.
  • È formato da coppie chiave => valore.
  • Puoi aggiungere, modificare e leggere i dati usando le chiavi.
  • Si usa foreach per leggere tutto il contenuto.

📌 Nota

Le chiavi possono essere:

  • Stringhe (es: "nome")
  • Numeri (es: 1, 100), ma a quel punto è come un array normale.

✅ Esempio completo

<?php
$libro = [
    "titolo" => "1984",
    "autore" => "George Orwell",
    "anno" => 1949
];

foreach ($libro as $campo => $info) {
    echo "$campo: $info\\n";
}
?>

Output:

titolo: 1984
autore: George Orwell
anno: 1949
